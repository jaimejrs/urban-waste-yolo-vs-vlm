# Gemma 4 31B-QAT Zero-Shot Prompt & Output Specification

This file contains the exact prompt, JSON schema, and robust parsing logic used to execute the zero-shot Gemma 4 31B-QAT experiments for the paper.

---

## 1. Verbatim Prompt (PROMPT_VLM)

```text
You are a public street monitoring analyst.
Your task is to answer one single question: IS THERE AT LEAST ONE VISIBLE GARBAGE BAG in this street image?

Count only bags that are visibly being used to contain discarded household or commercial waste.
The bag may be tied, open, full, partially full, isolated, or grouped with other garbage bags.

Do NOT count loose bottles, cans, cardboard, packaging, plastics or glass; empty shopping bags;
closed trash bins/containers in normal use; vehicles; people; vegetation; street furniture; or pavement.

Respond STRICTLY in valid JSON, with no markdown and no text outside the JSON.
Use exactly these field names:
{"tem_lixo": true, "descricao_breve": "...", "confianca": 0.0}

In this schema, tem_lixo means that at least one visible garbage bag is present.
If there is no visible garbage bag, return tem_lixo: false.
```

## 3. Robust JSON Parsing & Exception Handling (Python)

If the model output contains conversational text surrounding the JSON block or is enclosed in Markdown code fences (e.g., ```json ... ```), the following robust parser isolates and extracts the JSON object. 

If parsing fails or a request timeout occurs, the image is conservatively classified as having no visible garbage bag (`tem_lixo: false`).

```python
import json

def parse_json_robusto(raw: str) -> dict:
    if not raw:
        return {"parse_error": True}
    try:
        # Attempt direct JSON load
        return json.loads(raw)
    except json.JSONDecodeError:
        # Fallback: search for first '{' and last '}' to extract raw JSON
        i, j = raw.find("{"), raw.rfind("}") + 1
        if 0 <= i < j:
            try:
                return json.loads(raw[i:j])
            except json.JSONDecodeError:
                pass
    return {"parse_error": True}

def vlm_decisao(p: dict) -> int:
    # If a parsing error or network error occurred, treat as negative (no garbage bag)
    if p.get("parse_error") or p.get("erro"):
        return 0
    return int(bool(p.get("tem_lixo")))
```
